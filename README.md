# tracker-server

前端数据服务

### 开发

```bash
$ npm i
$ npm run dev
$ open http://127.0.0.1:7002/
```

### 发布

```bash
$ npm run release
```

### 部署

```bash
$ npm run pm2 // 启动
$ npm run restart // 重启
```
